# Aroma Coffee - Promo Site

Курсова робота на тему: **"Розробка промо-сайту для кав'ярні"**.
Веб-ресурс призначений для ознайомлення відвідувачів з меню, атмосферою закладу та надання можливості зворотного зв'язку.

## Функціонал
- **Адаптивний дизайн** (Mobile First).
- **Інтерактивне меню** з ефектами при наведенні.
- **Форма зворотного зв'язку** (AJAX запити без перезавантаження).
- **Збереження даних** у базу даних MySQL.

## Технології
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
- **Backend:** PHP (PDO), REST API approach.
- **Database:** MySQL.

## Структура проєкту
- `/api` — серверна логіка (підключення до БД, обробка пошти).
- `/assets` — стилі, скрипти та зображення.
- `index.html` — головна сторінка.
- `database.sql` — дамп бази даних для імпорту.
